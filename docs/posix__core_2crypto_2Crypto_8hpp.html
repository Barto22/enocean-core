Detailed Description

POSIX OpenSSL implementation of ICrypto<AesCcm> for AES-128-CCM-4.

Wraps the OpenSSL EVP AES-128-CCM API to provide the single aes_ccm_decrypt_impl() required by ICrypto<Derived>.

Requires: libssl-dev (apt install libssl-dev)
