Casts ctx to EnoceanDriver* and forwards to process_advertisement(). Suitable for platforms that require a plain function pointer with a void* context (e.g., the IBle::init(cb, ctx) overload).

BLE advertisement callback entry point.

Intended to be called from the BLE scanning thread for every received non-connectable advertisement. Thread-safety is the caller's responsibility (the BLE layer must serialise calls or acquire a lock).

Parameters
addr6-byte BLE address (LE byte order).
addr_typeAddress type: 0 = public, 1 = random.
adv_typeHCI advertisement event type (0x00 = ADV_IND, etc.).
rssiReceived signal strength in dBm.
dataRaw advertisement payload (max 31 bytes).
