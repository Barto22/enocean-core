Detailed Description

Zephyr RTOS watchdog driver implementation.

This file provides a concrete implementation of the IWatchdog interface for the Zephyr RTOS platform. It wraps Zephyr's native watchdog API to provide hardware watchdog timer functionality with a consistent interface.
