Detailed Description

POSIX-specific sleep implementation.

Zephyr-specific sleep implementation.

Implements the ISleep interface for POSIX platforms, providing thread suspension via POSIX mechanisms (e.g., nanosleep, usleep, etc).

Implements the ISleep interface for Zephyr platforms, providing thread suspension via Zephyr mechanisms (e.g., k_sleep).
