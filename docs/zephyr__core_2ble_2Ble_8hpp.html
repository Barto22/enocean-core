Detailed Description

Zephyr implementation of the BLE passive scanner interface.

Wraps the Zephyr Bluetooth observer (bt_le_scan_cb_register) and forwards each advertisement to the registered AdvCallback.

The application must call bt_enable(NULL) before calling init().
