Detailed Description

template<typename D, typename BufferType>
struct PlatformCore::QueueInitHelper< true, D, BufferType >

Helper that validates message-queue implementations requiring an external storage buffer.

When RequiresBuffer is true, this specialization checks that the derived type exposes an init_impl(BufferType*, uint32_t) signature returning an ErrorHandler via the QueueInitWithBuffer concept.
