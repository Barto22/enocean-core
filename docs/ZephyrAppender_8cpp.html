Detailed Description

plog-to-Zephyr logging bridge implementation.

This translation unit owns the Zephyr log module registration so that all plog output appears under a single "app" module in the Zephyr shell/RTT. The write() method is called by plog from any thread; Zephyr's logging subsystem is itself thread-safe.
