Detailed Description

Captures call-site source location and dispatches to plog.

Constructing this struct in a macro causes std::source_location::current() to be evaluated at the macro expansion point (the actual call site), giving plog accurate file/line/function information.

operator() formats the printf-style message into a fixed-size stack buffer and submits a plog::Record with the stored source location.
