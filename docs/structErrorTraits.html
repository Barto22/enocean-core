Detailed Description

template<ErrorEnum E>
struct ErrorTraits< E >

Traits for converting error enum values to human-readable names.

Users must specialize this struct for each error enum type to provide meaningful error messages. The specialization should implement a static name() function that returns a string_view describing the error.

Template Parameters
EThe error enum type (must satisfy ErrorEnum concept)
Note
Specializations should be constexpr for compile-time evaluation
Warning
Default implementation returns "Unknown Error" - always specialize!
