Detailed Description

POSIX-specific GPIO driver implementation.

Zephyr-specific GPIO driver implementation.

Implements the IGpio interface for POSIX platforms, providing digital I/O control via file descriptors and system calls.

Implements the IGpio interface for Zephyr platforms, providing digital I/O control via Zephyr device tree and driver APIs.
