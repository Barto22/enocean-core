Detailed Description

Concept to validate watchdog implementation requirements.

Ensures that a type T provides the required interface methods for watchdog functionality: init_impl() returning bool and feed_impl() returning void. The type must also be a concrete (non-abstract) class.

Template Parameters
TThe type to check for watchdog implementation compliance.
