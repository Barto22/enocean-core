Detailed Description

CRTP interface for platform-specific BLE passive scanners.

Provides a compile-time polymorphic contract for BLE scanning, consistent with the IThread / IMessageQueue pattern used throughout the project.

Platforms implement this interface by deriving from IBle<Derived> and providing the three _impl methods listed in the BleImplementation concept.
