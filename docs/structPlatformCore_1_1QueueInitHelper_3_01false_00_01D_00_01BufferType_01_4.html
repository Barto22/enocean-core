Detailed Description

template<typename D, typename BufferType>
struct PlatformCore::QueueInitHelper< false, D, BufferType >

Helper that validates message-queue implementations managing their own storage internally.

When RequiresBuffer is false, this specialization verifies that the derived type provides an init_impl(uint32_t) function returning an ErrorHandler using the QueueInitWithoutBuffer concept.
