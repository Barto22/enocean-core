Detailed Description

POSIX-specific thread implementation.

Zephyr-specific thread implementation.

Implements the IThread interface for POSIX platforms, providing thread creation, management, and destruction via POSIX mechanisms (e.g., pthreads).

Implements the IThread interface for Zephyr platforms, providing thread creation, management, and destruction via Zephyr mechanisms.
