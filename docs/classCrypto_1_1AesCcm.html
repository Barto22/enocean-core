Detailed Description

AES-128-CCM-4 backend using OpenSSL EVP_CIPHER_CTX.

AES-128-CCM-4 backend using Zephyr PSA AEAD API.

Stateless: every call to aes_ccm_decrypt_impl() creates and frees a temporary EVP_CIPHER_CTX on the stack (via RAII wrapper), so no persistent state is needed and the class is trivially default constructible.

AES-128-CCM-4 tag verification (+ optional decryption).

Verifies the CCM authentication tag and optionally decrypts ciphertext. If ciphertext is empty, performs tag-only verification (authentication without decryption). Uses OpenSSL EVP API for the cryptographic operations.

Parameters
key16-byte AES-128 key span
nonce13-byte CCM nonce span
aadAdditional authenticated data (may be empty)
ciphertextEncrypted payload to decrypt (may be empty for auth-only)
tag4-byte CCM authentication tag to verify
plaintextOutput buffer for decrypted data (may be empty if no ciphertext)
Returns
true if tag verification succeeds, false otherwise
