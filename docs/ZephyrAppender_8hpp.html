Detailed Description

plog appender that routes log records to the Zephyr logging subsystem.

The appender implementation lives in ZephyrAppender.cpp which owns the LOG_MODULE_REGISTER declaration required by Zephyr. All plog log records produced anywhere in the application are funnelled through this single appender and re-emitted via the appropriate Zephyr LOG_xxx macro.

Level mapping: plog::fatal / plog::error -> LOG_ERR plog::warning -> LOG_WRN plog::info -> LOG_INF plog::debug / plog::verbose -> LOG_DBG
